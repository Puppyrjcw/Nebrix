***

## Getting Started

Currently Nebrix is only supported on **Windows with Visual Studio 2022**.

Before downloading Nebrix, make sure you have:

- **Visual Studio 2022** with the *Desktop development with C++* workload
- A **GPU that supports OpenGL 3.3 or higher**

#### Installation

You can either download the .exe file in the latest releases or Build the project using Visual Studio 2022.

As of now Nebrix does not support CMake yet. Instead follow these steps:
1.  Download the Source code of the latest release.
2. Unzip the folder and open the .slnx file (Ensuring you have Visual Studio 2022+ installed)
3.  Make sure you run the **Debug Build**and click build project and open the .exe file located in Nebrix\x64\Debug
