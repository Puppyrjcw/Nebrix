# Nebrix
***
[![Nebrix Logo](https://i.postimg.cc/j2V0WgK9/Nebrix.png "Nebrix Logo")](https://i.postimg.cc/j2V0WgK9/Nebrix.png "Nebrix Logo")

Nebrix is a **work in progress** online platform where anyone can **create, share, and play games**. With powerful creation tools and a community-driven world, **the only limit is your imagination.**

***

## Getting Started

Currently Nebrix is only supported on **Windows with Visual Studio Installed**.

Before downloading Nebrix, make sure you have:

- **Visual Studio 2022** with the *Desktop development with C++* workload (Including some required .dlls that come with Visual Studio)
- A **GPU that supports OpenGL 3.3 or higher**

## Installation

1. Download the latest release `.exe` from the GitHub Releases page  
2. Run the executable  
3. No installation is required (portable build)

> An installer is currently in development to remove the requirement for Visual Studio.

## Future

Nebrix is still in early development, but we have exciting plans for the future:

- Multiplayer & Servers: We plan to add dedicated servers so you can play and collaborate with friends in real-time.
- Custom Avatars: Soon, players will be able to create and customize their own avatars.
- Custom Assets: Users will be able to import custom meshes, textures, and other assets to bring their creations to life.
- Expanded Creation Tools: More advanced building, scripting, and gameplay options will allow endless creativity.
